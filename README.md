# Project Overview

## But de l'application

TV Series Manager est une application web permettant aux utilisateurs de découvrir de nouvelles séries à regarder, de retrouver celles qu'ils aiment, de garder une trace de leur progression de visionnage d'une série, et de gérer leurs favoris.

## Fonctionnalités principales

* Recherche de séries TV (avec filtres avancées)
* Page détail d'une série (saisons, épisodes, synopsis, ...)
* Liste de séries populaires
* Gestion locale des favoris (Ajout / Suppression) et de leur visionnage

## Stack technique

* Next.js / React
* TypeScript
* Tailwind CSS
* The Movie DataBase API
* FontAwesome

## Objectif d'apprentissage

L'objectif était de démontrer ma capacité à concevoir une application front complète, en illustrant ma maîtrise de React/TS, du state management, des appels API, du routing, et de la création de composants réutilisables.
