# TV Series Manager (en cours de développement) - Project Overview

## But de l'application

TV Series Manager est une application web permettant aux utilisateurs de découvrir de nouvelles séries à regarder, de retrouver celles qu'ils aiment, de gérer leurs favoris et leur watchlist.

## Fonctionnalités principales

* Recherche de séries TV (avec filtres)
* Page détail d'une série (synopsis, nombres de saisons et d'épisodes, statut, ...)
* Liste de séries populaires
* Gestion locale des favoris (Ajout / Suppression) et d'une watchlist

## Stack technique

* Next.js / React
* TypeScript
* Tailwind CSS
* The Movie DataBase API
* React Context API
* FontAwesome

## Objectif d'apprentissage

L'objectif était de démontrer ma capacité à concevoir une application front complète, en illustrant ma maîtrise de React/TS, du state management, des appels API, du routing, et de la création de composants réutilisables.
